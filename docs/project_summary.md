# Project Summary

## Background

Healthcare organizations rely on revenue cycle data to understand billing performance, payer activity, provider productivity, patient volumes, and outstanding balances.

This project uses Microsoft SQL Server and a simulated healthcare database to analyze financial and operational questions related to healthcare revenue cycle management.

## Project Objectives

The objectives of this project are to:

- Analyze patient and provider activity
- Evaluate CPT code utilization
- Measure Gross Collection Rate by healthcare location
- Compare payer and payment performance
- Identify credentialing-related write-offs
- Analyze physician specialty payments
- Calculate charge and payment per CPT unit
- Evaluate Net Collection Rate and Accounts Receivable
- Create location-level financial summaries
