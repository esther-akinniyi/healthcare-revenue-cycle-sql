# Healthcare Revenue Cycle Analytics Using SQL Server

## Project Overview

This project demonstrates healthcare revenue cycle and financial analytics using Microsoft SQL Server and a simulated relational healthcare database.

The analysis examines patient and provider activity, CPT code utilization, payer performance, physician specialty payments, credentialing-related write-offs, patient demographics, charge and payment rates, net collection rates, accounts receivable, and location-level financial performance.

## Key Findings

- Twin Mountains Hospital had the highest Gross Collection Rate at approximately 63.4%.
- Internal Medicine received the highest total payments.
- Angelstone Community Hospital had the largest credentialing-related write-offs.
- Obstetrics & Gynecology had the lowest Net Collection Rate among specialties with Net Charges greater than \$25,000.
- Approximately 83% of Net Charges were collected, 16% remained in Accounts Receivable, and approximately 1% was written off for other adjustment reasons.
